再来看看 String 的几个方法:

从上面三个方法可以看的出 , 无论是subString还是replace亦或是concat都是重新创建了一个String字符串 , 并没有对原字符串进行修改.

在这永远要记住一点：对String对象的任何改变都不会影响到原对象，相关的任何更改操作都会生成新对象。
